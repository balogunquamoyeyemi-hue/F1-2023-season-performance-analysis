# Formula 1 2023 Season Performance Analysis

**OVERVIEW**
This Project analyzes the 2023 Formula 1 Season, Focusing on driver performance, race outcomes, and overall season trends
The dashboard highlights key metrics as wins, consistency, and differences between drivers and teams

**OBJECTIVES**
- Analyze driver performance across the season
- Evaluate race wins and consistency
- Compare team performance
- Identify dominant drivers and trends

- **KEY INSIGHTS**
- A small number of drivers dominated race wins
- Consistency played a major role in overall standings
- Team performance significantly influenced results
- Performance gaps between were evident throughout the season

- **TOOLS USED**
- POWER BI
- Power Query ( Data Cleaning )
- DAX ( Measures & KPIs )
